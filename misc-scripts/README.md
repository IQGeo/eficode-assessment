# Misc scripts

This directory contains various scripts that are useful for development and testing.
Could be used for inspiration or as a starting point for your own scripts.

## List of scripts

- [**`export-repos.sh`**](export-repos.sh) - Pull repos into a csv file.
- [**`export-repos-with-properties.sh`**](export-repos-with-properties.sh) - Pull repos owners into a csv file.
- [**`export-repos-with-owners.sh`**](export-repos-with-owners.sh) - Pull repos custom properties into a csv file.
- [**`link-gh-teams.sh`**](link-gh-teams.sh) - Use gei to create-teams and link them to IdP groups using a provided mapping Excel file.
- [**`migrate.sh`**](migrate.sh) - Use gei to generate a migration powershell script from one org to another, using a provided mapping Excel file and an option to filter by a status column
